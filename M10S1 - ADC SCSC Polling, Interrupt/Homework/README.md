## Question
Bài tập thực hành kết hợp ADC và PWM:
Điều chỉnh biến trở (ADC) để thay đổi độ sáng LED (TIMER-PWM). Khi điều chỉnh biến trở về hết 1 hướng - LED OFF, khi điều chỉnh biến trở về hướng ngược lại - LED ON 100%.
